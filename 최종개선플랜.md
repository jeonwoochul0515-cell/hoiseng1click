## TIER 3: 중기 (2~4주) — 디자인 시스템 정비

### 3-1. 색상 체계 통일

**문제: Gold 색상을 3가지 방식으로 표현 중**
```
#C9A84C (하드코딩) — 대부분 파일
var(--color-brand-gold) (CSS 변수) — CollectionPage, StatementPage
amber-400 (Tailwind 기본) — App.tsx 스피너
```

**해결: Tailwind v4 `@theme` 클래스 통일**
```
bg-[#C9A84C] → bg-brand-gold
text-[#C9A84C] → text-brand-gold  
border-[#C9A84C] → border-brand-gold
```

**추가 색상 변수 등록:**
```css
@theme {
  --color-champagne: #E8D5A3;    /* gold 연한 배경 */
  --color-warm-white: #FFF8EE;   /* gold 계열 카드 배경 */
  --color-light-navy: #1E3A5F;   /* navy hover */
}
```

### 3-2. 버튼 시스템 정리

**현재 5종 혼재 → 3종으로 정리:**

| 역할 | 클래스 | 용도 |
|------|--------|------|
| **Primary** | `bg-brand-gold text-black font-bold rounded-lg` | CTA, 확인, 저장 |
| **Secondary** | `border border-gray-300 text-gray-700 rounded-lg` | 취소, 보조 액션 |
| **Danger** | `bg-red-600 text-white rounded-lg` | 삭제, 위험 액션 |

- `bg-blue-600`, `bg-indigo-600` → 모두 Primary(gold)로 교체
- ClientsPage 의뢰인 등록 버튼: `bg-blue-600` → `bg-brand-gold`

### 3-3. 카드 시스템 통일

| 패턴 | 클래스 |
|------|--------|
| 기본 카드 | `rounded-xl bg-white border border-gray-200 p-6` |
| 모달 | `rounded-2xl bg-white p-8 shadow-xl` |
| 강조 카드 | `rounded-xl bg-white border-2 border-brand-gold p-6` |

### 3-4. STATUS_COLORS 중앙 집중

```typescript
// src/constants/status.ts (신규 생성)
export const STATUS_COLORS = {
  new:        { bg: 'bg-blue-500/20', text: 'text-blue-500', dot: 'bg-blue-500' },
  contacted:  { bg: 'bg-purple-500/20', text: 'text-purple-500', dot: 'bg-purple-500' },
  collecting: { bg: 'bg-brand-gold/20', text: 'text-brand-gold', dot: 'bg-brand-gold' },
  drafting:   { bg: 'bg-purple-500/20', text: 'text-purple-500', dot: 'bg-purple-500' },
  submitted:  { bg: 'bg-amber-500/20', text: 'text-amber-500', dot: 'bg-amber-500' },
  approved:   { bg: 'bg-emerald-500/20', text: 'text-emerald-500', dot: 'bg-emerald-500' },

```

### 3-5. Input 스타일 통일

```
모든 Input: border border-gray-300 px-4 py-2.5 rounded-lg text-sm
            focus:border-brand-gold focus:ring-1 focus:ring-brand-gold outline-none
```

- ClientsPage 검색 `focus:border-blue-500` → `focus:border-brand-gold`
- SettingsPage `border-gray-200` → `border-gray-300`
- LiquidationPage label `text-xs` → `text-sm`

### 3-6. Sidebar 배경색 수정

```
현재: bg-[#111827] (gray-900)
CSS 변수: --color-bg-sidebar: #1E293B
→ Sidebar.tsx를 bg-bg-sidebar로 통일하거나, CSS 변수를 #111827로 맞추기
```
