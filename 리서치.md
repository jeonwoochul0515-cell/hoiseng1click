## 7. 간이양식 vs 정식양식 분석

> 조사일: 2026-04-13

### 7-1. 핵심 차이

| 구분 | 간이양식 | 정식양식 |
|------|---------|---------|
| 대상 | 단순 채무 (금융채무, 급여소득자) | 복잡한 채무 (사업소득, 담보채권) |
| 변제 방식 | 가용소득만으로 변제 | 가용소득 + 재산처분 포함 |
| 분량 | 간소 (4~6p) | 상세 (10p+) |
| 보정 요구 | 적음 | 많을 수 있음 |
| 사용 비율 | 약 70~80% (대부분) | 약 20~30% |

### 7-2. 서류별 차이

| 서류 | 간이양식 | 정식양식 차이점 |
|------|---------|---------------|
| 신청서 (간이양식 1) | 기본 인적사항 + 채무총액 | 동일 구조 |
| 채권자목록 (간이양식 1-1) | 5열 간소 구조 | 발생원인, 이자율 구분, 담보 상세 등 11열+ |
| 재산목록 (간이양식 1-2) | 주요 재산만 기재 | 세부 항목별 감정가, 처분 계획 포함 |
| 수입지출목록 (간이양식 1-3) | 기준중위소득 60% 체크박스 | 항목별 상세 금액, 증빙 목록 |
| 진술서 (간이양식 1-4) | 채무 경위 서술 | 동일 구조 |
| 변제계획안 (간이양식 2) | 가용소득 기준 단순 계산 | 재산처분 일정 + 배당표 + 별제권 처리 포함 |

### 7-3. 간이양식 사용 조건

- 급여소득자로 가용소득이 명확히 산정되는 경우
- 채무가 단순 금융채무 위주 (사업 관련 복잡한 채무 제외)
- 보유 재산이 소액 (별도 처분 계획 불필요)
- 채권자 수가 적고 채무 종류가 단순할 때

### 7-4. 정식양식이 필요한 경우

- 부동산 등 재산이 있어 처분 계획이 필요한 경우
- 별제권(담보채권)이 포함된 경우
- 사업소득자 (소득 증빙이 복잡)
- 채권자 수가 많고 채무 성격이 다양한 경우
- 법원 보정 명령으로 정식양식 교부받은 경우

### 7-5. 현재 프로젝트 구현 상태

| 항목 | 상태 | 비고 |
|------|------|------|
| DocPreview (미리보기) | 간이양식 6종 완전 구현 | 법원 양식과 레이아웃 일치 |

### 7-6. 법원별 양식 현황 (2026.3 기준)

- 서울회생법원: 간이양식 + 정식양식 모두 제공
- 부산회생법원: 동일 양식 체계
- 대구회생법원 (2026.3.1 신설): 동일 양식 체계
- 대전회생법원 (2026.3.1 신설): 동일 양식 체계
- 광주회생법원 (2026.3.1 신설): 동일 양식 체계
